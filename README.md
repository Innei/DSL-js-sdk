# Overview

The JS SDK for rss3 services.

## Usage

Install the package:

```bash
npm i @rss3/js-sdk
```

Basic code example: [link](examples/basic/index.ts).

For more examples see [examples](examples).

## Development

Install [use-node](https://github.com/ysmood/use-node).

```bash
use-node
npm i
npm t
```
